# Lineform Help

Lineform is a native Mac Markdown editor focused on calm writing, readable typography, local files, and accessible reading controls.

## Core Tasks

- Open and save `.md`, `.markdown`, and `.txt` files.
- Switch between Write, Read, and Split modes.
- Toggle the outline when headings are useful.
- Use Reading Experience to adjust type, spacing, themes, focus, ruler, and caret width.
- Use native Writing Tools or Lineform intelligence from the Write-mode AI rail, menus, or keyboard shortcuts when available.

## Local-First Privacy

Documents stay wherever the user saves them. Lineform does not add accounts, analytics, document upload, or external AI services.
