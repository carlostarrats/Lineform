# Lineform Help

Lineform is a free native Mac Markdown editor focused on calm writing, readable typography, local files, and accessible reading controls.

## Core Tasks

- Open and save `.md`, `.markdown`, and `.txt` files.
- Switch between Write, Read, and Preview modes.
- Toggle the outline when headings are useful.
- Search with match highlighting and next/previous navigation.
- Format Markdown headings, emphasis, inline code, lists, and links.
- Convert between Markdown and plain text.
- Export to PDF.
- Use Reading Experience to adjust type, spacing, themes, reading ruler, and typewriter mode.
- Use native Writing Tools or toggle AI in Write mode, select text, type an instruction in the AI bar, and review before accepting when available.

## Local-First Privacy

Documents stay wherever the user saves them. Lineform does not add accounts, analytics, document upload, or external AI services.
